# SCT_WD_4
A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript to add, delete, and manage daily tasks.
📝 To-Do List Web Application

This project is a To-Do List Web Application developed using HTML, CSS, and JavaScript.
It helps users manage their daily tasks by adding, completing, and deleting tasks easily.

---

🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and user-friendly interface
- Responsive design for different screen sizes

---

🛠 Technologies Used

- HTML – Structure of the application
- CSS – Styling and layout
- JavaScript – Task management and interactivity
